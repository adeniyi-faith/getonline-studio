( THE BRIEF )

The Challenge: Design a landing page for a next-generation digital finance protocol that moves away from boring, traditional bank aesthetics.

The Goal: Create a brand that feels mysterious, exclusive, and futuristic. The site serves as a pre-launch hub to capture early investor interest through an "Alpha Access" waitlist.

Interactive Atmosphere

The background gradient subtly shifts based on mouse movement, creating a living, breathing environment that reacts to the user.

📱

Mobile-First HUD

Content cards use glassmorphism to feel like Heads-Up Display (HUD) elements. On mobile, they stack vertically for seamless readability.

🔒

Waitlist System

A fully functional "Alpha Access" form designed to build hype. It collects emails and segments users based on investor type.
